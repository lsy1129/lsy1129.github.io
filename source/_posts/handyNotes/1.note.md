> 随手记记，方便回忆
### hexo发布静态资源

~~~ vim
hexo clean && hexo g && hexo d
~~~

### hexo生成文件

~~~ vim
hexo n 文件名
~~~

### windows下解压tar.gz文件

> 1. 搜索Windows PowerShell并打开
> 2. cd 到压缩文件所在文件夹
> 3. 然后使用命令 tar -zxvf 要解压的文件名

## 拉取远端指定分支并在本地新建分支

```bash
git checkout -b 本地分支 origin/远程分支
```
